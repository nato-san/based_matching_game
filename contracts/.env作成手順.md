# .envファイルの作成手順（詳しく）

## 📍 ファイルを作る場所

`.env`ファイルは、**`contracts`フォルダの中**に作る必要があります。

正しい場所：
```
Desktop/based_matching_game/contracts/.env
```

---

## 🖥️ Macの場合

### 方法1: ターミナルで作成（推奨）

1. **ターミナルを開く**
   - `Command + スペース` → 「ターミナル」と入力

2. **contractsフォルダに移動**
   ```bash
   cd Desktop/based_matching_game/contracts
   ```

3. **.envファイルを作成**
   ```bash
   nano .env
   ```

4. **内容を入力**
   ```
   PRIVATE_KEY=ここに秘密鍵を貼り付け
   BASE_RPC_URL=https://mainnet.base.org
   ```
   
   **例:**
   ```
   PRIVATE_KEY=あなたの秘密鍵
   BASE_RPC_URL=https://mainnet.base.org
   ```

5. **保存する**
   - `Ctrl + X` を押す
   - `Y` を押す（保存するか聞かれたら）
   - `Enterキー` を押す

### 方法2: テキストエディタで作成

1. **Finderでcontractsフォルダを開く**
   - `Desktop/based_matching_game/contracts`

2. **テキストエディタを開く**
   - メモ帳やVS Codeなど

3. **内容を入力**
   ```
   PRIVATE_KEY=ここに秘密鍵を貼り付け
   BASE_RPC_URL=https://mainnet.base.org
   ```

4. **保存する**
   - `Command + S` を押す
   - ファイル名を **`.env`** にする（先頭のドットを忘れずに！）
   - 保存場所を **`contracts`フォルダ**にする

---

## 💻 Windowsの場合

### 方法1: コマンドプロンプトで作成（推奨）

1. **コマンドプロンプトを開く**
   - `Windows + R` → 「cmd」と入力

2. **contractsフォルダに移動**
   ```bash
   cd Desktop\based_matching_game\contracts
   ```

3. **.envファイルを作成**
   ```bash
   notepad .env
   ```

4. **内容を入力**
   ```
   PRIVATE_KEY=ここに秘密鍵を貼り付け
   BASE_RPC_URL=https://mainnet.base.org
   ```

5. **保存する**
   - `Ctrl + S` を押す
   - メモ帳を閉じる

### 方法2: メモ帳で直接作成

1. **contractsフォルダを開く**
   - `Desktop\based_matching_game\contracts`

2. **メモ帳を開く**

3. **内容を入力**
   ```
   PRIVATE_KEY=ここに秘密鍵を貼り付け
   BASE_RPC_URL=https://mainnet.base.org
   ```

4. **保存する**
   - `Ctrl + S` を押す
   - ファイル名を **`.env`** にする（先頭のドットを忘れずに！）
   - 「ファイルの種類」を「すべてのファイル」にする
   - 保存場所を **`contracts`フォルダ**にする

---

## ✅ 確認方法

ターミナル（コマンドプロンプト）で、以下を実行：

```bash
cd Desktop/based_matching_game/contracts
ls -la .env
```

または

```bash
cd Desktop\based_matching_game\contracts
dir .env
```

ファイルが表示されれば、正しく作成されています！

---

## ⚠️ よくある間違い

### ❌ 間違い1: ファイル名が違う
- `.env.txt` → ❌
- `env` → ❌
- `.env` → ✅

### ❌ 間違い2: 場所が違う
- `Desktop/based_matching_game/.env` → ❌
- `Desktop/based_matching_game/contracts/.env` → ✅

### ❌ 間違い3: 秘密鍵の前にスペースがある
- `PRIVATE_KEY= 0x123...` → ❌（スペースがある）
- `PRIVATE_KEY=0x123...` → ✅（スペースなし）
