# デプロイに必要なファイル

## ✅ デプロイに必要なファイル

### 必須ファイル

1. **`index.html`** - メインのゲームファイル（必須）

2. **`images/` フォルダ** - 画像と音声ファイル（必須）
   - `base_top.JPG` - TOP画像
   - `base_00.jpg` - カードの裏面
   - `base01.JPG` ～ `base15.jpg` - ゲームで使用する画像
   - `flip.mp3` - カードをめくる音
   - `match.mp3` - マッチしたときの音
   - `share.jpg` - シェア画像

3. **`.well-known/farcaster.json`** - Farcaster mini appの設定ファイル（必須）
   - Farcaster mini appとして動作するために必要

### オプションファイル（ある場合）

4. **`vercel.json`** - Vercelの設定ファイル（Vercelを使う場合）
5. **`netlify.toml`** - Netlifyの設定ファイル（Netlifyを使う場合）
6. **`_redirects`** - リダイレクト設定（Netlifyを使う場合）

---

## ❌ デプロイに不要なファイル

以下のファイルは**絶対にアップロイしないでください**：

1. **`contracts/` フォルダ** - コントラクトのソースコード（既にデプロイ済み）
2. **`.env` ファイル** - 秘密鍵が含まれている（絶対にアップロイしない！）
3. **`node_modules/` フォルダ** - 依存関係（不要）
4. **`.git/` フォルダ** - Gitの履歴（不要）
5. **`README.md`** などのドキュメントファイル（オプション）

---

## 📋 デプロイするファイル一覧

### 最小構成（必須）

```
based_matching_game/
├── index.html                    ← 必須
├── images/                       ← 必須
│   ├── base_top.JPG
│   ├── base_00.jpg
│   ├── base01.JPG
│   ├── base02.jpg
│   ├── ... (base03.jpg ～ base15.jpg)
│   ├── flip.mp3
│   ├── match.mp3
│   └── share.jpg
└── .well-known/                  ← 必須
    └── farcaster.json
```

### 完全構成（推奨）

```
based_matching_game/
├── index.html
├── images/
│   └── (上記と同じ)
├── .well-known/
│   └── farcaster.json
├── vercel.json                   ← Vercelを使う場合
└── _redirects                    ← Netlifyを使う場合
```

## 🚀 デプロイ方法

### Vercelの場合

1. **GitHubにプッシュ**（推奨）
   - 必要なファイルだけをコミット
   - GitHubにプッシュ
   - VercelでGitHubリポジトリを接続

2. **直接アップロード**
   - Vercelのダッシュボードで「Add New Project」
   - 「Upload」を選択
   - 必要なファイルを選択してアップロード

### Netlifyの場合

1. **GitHubにプッシュ**（推奨）
2. **直接アップロード**
   - Netlifyのダッシュボードで「Add new site」
   - 「Deploy manually」を選択
   - 必要なファイルをドラッグ&ドロップ

---

## ⚠️ 重要な注意事項

### 絶対にアップロイしないファイル

1. **`.env` ファイル**
   - 秘密鍵が含まれている
   - アップロイすると、秘密鍵が公開されてしまいます
   - **絶対にアップロイしないでください！**

2. **`contracts/` フォルダ**
   - コントラクトは既にデプロイ済み
   - ソースコードは不要

3. **`node_modules/` フォルダ**
   - サイズが大きい
   - 不要

- [ ] コントラクトアドレスが正しく設定されているか確認

## 💡 ヒント

### .gitignoreファイルを使う

`.gitignore`ファイルを作成して、不要なファイルを除外：

```
.env
node_modules/
contracts/
.git/
```

これで、GitHubにプッシュするときに自動的に除外されます。
