# コンソールのエラーを確認する方法

## 🎯 優先順位

たくさんのエラーがある場合、以下の順番で確認してください：

### 1. 最重要：Farcaster SDK関連のエラー

**探すべきキーワード：**
- `Farcaster`
- `ready`
- `sdk`
- `Ready not called`

**例：**
```
❌ Ready not called
❌ Farcaster SDK not found
❌ sdk.actions.ready is not a function
```

### 2. 重要：ethers関連のエラー

**探すべきキーワード：**
- `ethers`
- `ethers is not defined`
- `ProviderError`

**例：**
```
❌ ethers is not defined
❌ ProviderError: insufficient funds
```

### 3. その他のエラー

- ネットワークエラー（CORS、404など）
- 画像の読み込みエラー
- その他のJavaScriptエラー

---

## 📋 確認手順

### ステップ1: Farcaster SDK関連のエラーを探す

1. コンソールを開く（`F12`）
2. コンソール上部の検索ボックスに「**Farcaster**」と入力
3. または「**ready**」と入力
4. 表示されたエラーメッセージをコピー

### ステップ2: エラーメッセージをコピー

**方法1: エラーメッセージをクリック**
- エラーメッセージをクリックすると、詳細が表示されます
- そのメッセージ全体をコピー

**方法2: 右クリックでコピー**
- エラーメッセージを右クリック
- 「コピー」を選択

### ステップ3: 最初の3-5個のエラーを伝える

最も上に表示されているエラーから、最初の3-5個を伝えてください。

---

## 🔍 具体的な確認方法

### 方法1: フィルターを使う

1. コンソール上部の「フィルター」アイコンをクリック
2. 「Errors」のみを表示
3. エラーのみが表示されます

### 方法2: 検索を使う

1. コンソール上部の検索ボックス（🔍）をクリック
2. 以下のキーワードで検索：
   - `Farcaster`
   - `ready`
   - `sdk`
   - `ethers`

### 方法3: スクリーンショットを撮る

1. コンソールを開く
2. エラーが表示されている部分をスクリーンショット
3. 画像を送る

---

## 📝 伝えるべき情報

以下の情報を教えてください：

### 1. 最も上に表示されているエラー（最初の3-5個）

**例：**
```
❌ Ready not called
❌ Farcaster SDK not found
⚠️ Farcaster SDK ready() error: ...
```

### 2. 「Farcaster」または「ready」を含むエラー

**例：**
```
❌ sdk.actions.ready is not a function
⚠️ Farcaster SDK not available after waiting
```

### 3. エラーの詳細（可能であれば）

エラーメッセージをクリックすると、詳細が表示されます。
その詳細も含めて伝えてください。

---

## 🎯 最も重要なエラーから確認

以下の順番で確認してください：

### 1. 「Ready not called」エラー

これが最も重要です。このエラーがあるか確認してください。

### 2. 「Farcaster SDK」関連のエラー

- `Farcaster SDK not found`
- `Farcaster SDK ready() error`
- `sdk.actions.ready is not a function`

### 3. 「ethers」関連のエラー

- `ethers is not defined`
- `ProviderError`

---

## 💡 簡単な方法

**最も簡単な方法：**

1. コンソールを開く（`F12`）
2. コンソール上部の検索ボックスに「**Farcaster**」と入力
3. 表示されたエラーメッセージをすべてコピー
4. それをそのまま伝える

または：

1. コンソールを開く（`F12`）
2. エラーが表示されている部分をスクリーンショット
3. 画像を送る

---

## 📸 スクリーンショットの撮り方

### Macの場合

1. `Shift + Command + 4` を同時に押す
2. マウスでコンソールのエラー部分を選択
3. スクリーンショットが保存されます

### Windowsの場合

1. `Windows + Shift + S` を同時に押す
2. マウスでコンソールのエラー部分を選択
3. クリップボードにコピーされます

---

1. コンソールを開く（`F12`）

3. 表示されたエラーメッセージをコピーして伝える

または：

1. コンソールを開く（`F12`）
2. エラー部分をスクリーンショット
3. 画像を送る
